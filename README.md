# jaani-builds.github.io

Root redirect site for GitHub Pages.

This repository redirects https://jaani-builds.github.io to https://jaani-builds.github.io/resume/.

## Publish

1. Create a new GitHub repository named jaani-builds.github.io.
2. Push this folder contents to the main branch of that repository.
3. In repository settings, enable GitHub Pages from main branch root.

After deployment, visiting the root URL will automatically forward to /resume/.
